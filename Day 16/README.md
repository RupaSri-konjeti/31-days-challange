## Day 16 – Linked List Basics

Today's focus was building a strong foundation in Linked Lists by implementing the core operations from scratch in Java.

### Problems Solved

* Create a Linked List and print all elements
* Insert a node at the beginning
* Insert a node at the end

### Key Learnings

* Understood the structure of a Node and how nodes are connected.
* Learned the role of the `head` pointer in managing the list.
* Practiced traversal and insertion operations without using Java's built-in LinkedList class.
* Strengthened my understanding of pointer manipulation and dynamic data structures.

Looking forward to exploring more Linked List operations in the next phase of the challenge.
