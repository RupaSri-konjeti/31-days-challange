# Day 14 - Sliding Window Technique

## Topics Covered

* Sliding Window Pattern
* Fixed-Size Sliding Window
* Time Complexity Optimization (O(n²) → O(n))

## Problems Solved

1. Maximum Sum Subarray of Size K
2. Average of All Subarrays of Size K
3. Maximum Number of Vowels in a Substring of Length K

## Key Learnings

* Learned how to efficiently process contiguous subarrays and substrings.
* Understood how maintaining a running window reduces redundant computations.
* Practiced identifying problems where the Sliding Window pattern can be applied.
* Improved optimization skills by converting brute-force approaches into linear-time solutions.
* Strengthened problem-solving skills for both array and string-based questions.

## Takeaway

Sliding Window is a powerful technique for solving subarray and substring problems efficiently. Mastering this pattern helps in tackling many interview questions involving arrays and strings.
