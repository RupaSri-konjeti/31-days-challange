# Day 15 - Kadane's Algorithm

## Topics Covered

* Maximum Subarray Sum
* Kadane's Algorithm
* Circular Subarray Sum
* Stock Buy and Sell for Maximum Profit

## Problems Solved

1. Maximum Subarray Sum
2. Maximum Circular Subarray Sum
3. Best Time to Buy and Sell Stock

## Key Learnings

* Learned how Kadane's Algorithm finds the maximum subarray sum in O(n) time.
* Understood the concept of maintaining a running sum and resetting it when it becomes negative.
* Explored the extension of Kadane's Algorithm for solving Circular Subarray problems.
* Applied similar optimization principles to the Best Time to Buy and Sell Stock problem.
* Improved understanding of array optimization techniques and linear-time algorithms.

## Takeaway

Kadane's Algorithm is one of the most important array algorithms for coding interviews. It demonstrates how maintaining optimal intermediate results can transform brute-force solutions into highly efficient O(n) approaches.
