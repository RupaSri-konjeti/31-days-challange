# Day 4/31 – Bubble Sort

Today I learned the fundamentals of Sorting and implemented Bubble Sort in Java.

### Topics Covered

* Why Sorting is Needed
* Ascending vs Descending Order
* Bubble Sort Algorithm
* Adjacent Element Comparison
* Time Complexity: O(n²)

### Practice Problems

* Sorted {5,4,3,2,1}
* Sorted {10,5,8,1,7}
* Performed manual dry runs to understand each pass and swap operation

### Key Takeaway

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. After each pass, the largest unsorted element moves to its correct position.
